# FIFA 온라인4 전적검색
### 개발기간 : 2022년 01월 24일 ~ 2022년 02월 02일

### [개요]

**FIFA 온라인4 전적검색 모바일 앱 개발**

Nexon 개발자센터에서 제공하는 Open Api를 사용한 모바일 전적검색 앱

* 유저정보 : 기본정보를 제공
* 랭크정보 : 최고등급 정보를 제공
* 구매기록 : 최근 구매목록 제공
* 판매기록 : 최근 판매목록 제공
* 최근전적 : 전적목록 제공
* 상세정보 : 사용한 선수목록 제공
